# ADHD特化型・強制着手タイマー (ADHD Timer App)

**👉 [今すぐWebブラウザで使う (Webアプリ版)](https://tagkaz.github.io/adhd-timer/)**

「タスクはあるのに、どれから始めればいいか迷ってしまって手がつかない…」
「一度集中するとやめられず、休憩を取るのを忘れてしまう…」

そんな悩みを解決するために作られた、**ADHD（注意欠如・多動症）の方に特化したポモドーロタイマー**です。
インストール等は一切不要で、上記のリンクからすぐに利用できます。（ローカルに保存して使える1ファイル版もあります）
## ✨ 主な機能

1. **「決められない時（開始）」ルーレット機能**
   - 複数のタスクを入力しておき、大きな赤いボタンを押すことで、ルーレット形式で強制的に最初のタスクを1つ決定します。「どれからやろう？」と迷う時間を物理的に排除します。

2. **大きな文字とハイコントラストな画面**
   - 視界に入る余計な情報を極力減らし、タイマーの数字と現在のタスク名だけを画面いっぱいに表示します。

3. **「次の一歩をメモして脳を空にする」機能（休憩前）**
   - 25分の作業が終わった後、作業のキリが悪くても「次に何をするか」をメモ画面に吐き出してから休憩に入れます。これにより、ワーキングメモリを解放してしっかり休むことができます。

4. **過集中（ハイパーフォーカス）防止アラーム**
   - 25分のタイマーがゼロになった後も、マイナス表示（超過時間）としてカウントアップを続けます。同時に専用のアラーム音が鳴り、強制的に作業を区切るよう促します。

5. **ブラウザへの自動保存**
   - 入力したタスクはブラウザ（LocalStorage）に自動保存されるため、次回開いたときもすぐ前回の続きから始められます。

## 🚀 使い方

### ローカルで使う場合
1. このリポジトリから `index_dist.html` をダウンロードします。
2. ダウンロードしたファイルをダブルクリックして、ChromeやSafariなどのWebブラウザで開くだけで動作します。



## 🛠️ 動作環境
- モダンブラウザ（Google Chrome, Apple Safari, Mozilla Firefox, Microsoft Edge）
- ※音声アラームを鳴らすため、ブラウザの音量設定をオンにしてご利用ください。
